Tổng quan

Zalo Java SDK  là một thư viện mã nguồn mở, hỗ trợ các nhà phát triển dễ dàng tích hợp đăng nhập Zalo và sử dụng các API của Zalo. Tham khảo cách lấy Access Token để sử dụng API của Zalo. Để được hướng dẫn cài đặt và triển khai, truy cập mục Bắt đầu nhanh và xem  một số ví dụ cụ thể, qua đó sẽ giúp bạn dễ dàng hơn trong việc tích hợp.
