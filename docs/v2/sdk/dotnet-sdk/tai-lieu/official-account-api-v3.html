Official Account API V3

Thông tin chi tiết về các API vui lòng tham khảo tài liệu Official Account API. Hướng dẫn gọi Official Account API sử dụng DotNet SDK.

Khai báo sử dụng Zalo DotNet SDK

using ZaloDotNetSDK;

Khởi tạo ZaloClient

ZaloClient client = new ZaloClient(access_token);

Gửi tin tư vấn dạng văn bản

JObject jObject = client.sendTextMessageToUserIdV3(user_id, "text_message");

Gửi tin tư vấn đính kèm hình ảnh

JObject jObject = client.sendImageMessageToUserIdByUrlV3(user_id, "text_message", "https://stc-developers.zdn.vn/images/bg_1.jpg");

Gửi tin Tư vấn theo mẫu yêu cầu thông tin người dùng

JObject jObject = client.sendRequestUserProfileToUserIdV3(user_id, "Official Account", "Đang yêu cầu thông tin từ bạn", "https://stc-developers.zdn.vn/images/bg_1.jpg");

Gửi tin Tư vấn đính kèm file

JObject jObject = client.sendFileToUserIdV3(user_id, "attachment_id");

Gửi tin Tư vấn trích dẫn

JObject jObject = client.sendQuoteMessageToUserIdV3(user_id, "text_messate", "quote_message_id");

Gửi tin Tư vấn kèm Sticker

JObject jObject = client.sendStickerMessageToUserIdV3(user_id, "sticker_id");

Gửi tin nhắn văn bản đến người dùng ẩn danh

JObject jObject = client.sendTextMessageToAnonyous("anonymous_id", "conversation_id", "text_message");

Gửi tin nhắn dạng ảnh đến người dùng ẩn danh

JObject jObject = client.sendImageMessageToAnonyousByUrl("anonymous_id", "conversation_id", "text_message", "https://stc-developers.zdn.vn/images/bg_1.jpg");

Gửi tin nhắn dạng file đến người dùng ẩn danh

JObject jObject = client.sendFileToAnonyous("anonymous_id", "conversation_id", "attachment_id");

Gửi tin nhắn dạng sticker đến người dùng ẩn danh

JObject jObject = client.sendStickerMessageAnonyous("anonymous_id", "conversation_id", "attachment_id");

Thả biểu tượng cảm xúc vào tin nhắn

JObject jObject = client.sendReactionToMessage(user_id, ReactIcon.HAHA.getValue(), "react_message_id");

Tải lên hình ảnh

JObject result = client.uploadImageForOfficialAccountAPI(new ZaloFile("C:\\Users\\CPU10932-local\\Desktop\\developers_zalo.png"));

Tải lên ảnh gif

JObject result = client.uploadGifForOfficialAccountAPI(new ZaloFile("C:\\Users\\CPU10932-local\\Desktop\\minion.gif"));

Tải lên file

ZaloFile file = new ZaloFile("C:\\Users\\CPU10932-local\\Desktop\\pdf_4mb.pdf");
file.setMediaTypeHeader("application/pdf");
JObject result = client.uploadFileForOfficialAccountAPI(file);

Kiểm tra hạn mức Tin tư vấn miễn phí

JObject jObject = client.getQuotaCSMessageFree();

Kiểm tra hạn mức Tin tư vấn 48h

JObject jObject = client.getQuotaCSMessageIn48h("message_id");

Kiểm tra hạn mức Tin truyền thông

JObject jObject = client.getQuotaPromotionMessage(user_id);
