Official Account API

Thông tin chi tiết về các API vui lòng tham khảo tài liệu Official Account API. Hướng dẫn gọi Official Account API sử dụng DotNet SDK. Khai báo sử dụng Zalo DotNet SDK

using ZaloDotNetSDK;

Khởi tạo ZaloClient​

ZaloClient client = new ZaloClient(access_token);

Lấy danh sách người quan tâm​

JObject result = client.getListFollower(0, 20);

Lấy thông tin người quan tâm​

JObject result = client.getProfileOfFollower("2468458835296197922");

Gửi tin nhắn cho user id​

JObject result = client.sendTextMessageToUserId("2468458835296197922", "this isn a message");

Gửi tin nhắn hình ảnh cho user id​

JObject result = client.sendImageMessageToUserIdByUrl("2468458835296197922", "this is a message", "https://stc-developers.zdn.vn/zalo.png");

Gửi tin nhắn hình ảnh cho user id bằng attachment id​

JObject result = client.sendImageMessageToUserIdByAttachmentId("2468458835296197922", "this is a messag", "a285de002301ca5f9310");

Gửi tin nhắn ảnh gif cho user id bằng attachment id​

JObject result = client.sendGifMessageToUserIdByAttachmentId("2468458835296197922", "this is a message", "KAyZCEjSFBaWeBw74O0OxX+w5DQ++3snJhi139F2zMk=");

Gửi tin nhắn list element cho user id​

List<Element> elements = new List<Element>();
OpenUrlElement openUrlElement = new OpenUrlElement("openUrlButton", "subtitle", "https://stc-developers.zdn.vn/zalo.png", "https://stc-developers.zdn.vn/zalo.png");
OpenSMSElement openSMSElement = new OpenSMSElement("openSMSButton", "subtitle", "https://stc-developers.zdn.vn/zalo.png", "0123456789", "message sms");
OpenPhoneElement openPhoneElement = new OpenPhoneElement("openPhoneButton", "subtitle", "https://stc-developers.zdn.vn/zalo.png", "0123456789");
QueryHideElement queryHideElement = new QueryHideElement("queryHideButton", "subtitle", "https://stc-developers.zdn.vn/zalo.png", "queryHideButton");
QueryShowElement queryShowElement = new QueryShowElement("queryShowButton", "subtitle", "https://stc-developers.zdn.vn/zalo.png", "queryShowButton");
elements.Add(openUrlElement);
elements.Add(openSMSElement);
elements.Add(openPhoneElement);
elements.Add(queryHideElement);
elements.Add(queryShowElement);
JObject result = client.sendListElementMessagetoUserId("2468458835296197922", "this is a message", elements);

Lấy danh sách nhãn​

JObject result = client.getAllTagOfOfficialAccount();

Gắn nhãn cho user​

JObject result = client.tagFollower("2468458835296197922", "tagname");

Gỡ nhãn khỏi user​

JObject result = client.removeTagFromFollower("2468458835296197922", "tagname");

Xòa nhãn​

JObject result = client.deleteTag("tagname");

Tải lên hình ảnh​

JObject result = client.uploadImageForOfficialAccountAPI(new ZaloFile("C:\\\\Users\\\\CPU10932-local\\\\Desktop\\\\developers_zalo.png"));

Tải lên ảnh gif​

JObject result = client.uploadGifForOfficialAccountAPI(new ZaloFile("C:\\\\Users\\\\CPU10932-local\\\\Desktop\\\\minion.gif"));

Tải lên file​

ZaloFile file = new ZaloFile("C:\\\\Users\\\\CPU10932-local\\\\Desktop\\\\pdf_4mb.pdf");
file.setMediaTypeHeader("application/pdf");
JObject result = client.uploadFileForOfficialAccountAPI(file);

Lấy thông tin của Official Account​

JObject result = client.getProfileOfOfficialAccount();

Lấy lịch sử chat gần nhất theo thời gian​

JObject result = client.getListRecentChat(0, 10);

Lấy lịch sử chat gần nhất theo user id​

JObject result = client.getListConversationWithUser(2468458835296197922, 0, 10);

Đăng ký ip​

JObject result = client.registerIP("ip", "some_name");

Hủy đăng ký ip​

JObject result = client.removeIP("ip", "some_name");

Cập nhật thông tin mở rộng của người quan tâm​

JObject result = client.updateFollowerInfo("2468458835296197922", "name", "84123456789", "address", 0, 0);

Broadcast tin nhắn danh sách element(chỉ hỗ trợ element oa.open.url)​

List<Element> elements = new List<Element>();
OpenUrlElement openUrlElement = new OpenUrlElement("openUrlButton", "subtitle", "https://stc-developers.zdn.vn/zalo.png", "https://stc-developers.zdn.vn/zalo.png");
elements.Add(openUrlElement);
elements.Add(openUrlElement);
JObject result = client.broadcastLinks(elements, TargetBroadcastGender.ALL);
