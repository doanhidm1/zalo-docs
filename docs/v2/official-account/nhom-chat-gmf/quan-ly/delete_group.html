OA Giải tán nhóm

Lưu ý

Ứng dụng cần được cấp quyền quản lý thông tin nhóm.

HTTP request​

  • URL: https://openapi.zalo.me/v3.0/oa/group/delete
  • Method: POST
  • Content Type: application/json
  • Response Type: text/json

Example request​

curl --location --request POST 'https://openapi.zalo.me/v3.0/oa/group/delete' \
--header 'access_token: <your_access_token>' \
--header 'Content-Type: application/json' \
--data '{
    "group_id": "f414c8f76fa586fbdfb4"
}'

Tham số header​

Tham sốKiểu dữ liệuTính bắt buộcMô tả
access_tokenstringyesToken cho phép ứng dụng đại diện OA gọi API. Xem chi tiết cách lấy access token  tại đây.

Cấu trúc body của request​

Tham sốKiểu dữ liệuTính bắt buộcMô tả
group_idstringyesID của nhóm

Example respond​

{
    "error": 0,
    "message": "Success"
}
