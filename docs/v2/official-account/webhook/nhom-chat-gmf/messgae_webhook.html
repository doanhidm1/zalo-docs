Sự kiện tin nhắn được gửi tới nhóm

Khi người dùng gửi tin nhắn đến nhóm, hệ thống Zalo sẽ gửi một request HTTP đến Webhook Url đã đăng ký của ứng dụng.

  • URL: Webhook Url của ứng dụng
  • Method: POST
  • Content Type: application/json
  • X-ZEvent-Signature: mac = sha256(appId + data + timeStamp + OAsecretKey), với data là chuỗi json trả về dưới đây

Example request​

{
  "event_name": "user_send_group_text",
  "app_id": "4591324301967398637",
  "sender": {
    "id": "8756287263669629130"
  },
  "recipient": {
    "id": "f414c8f76fa586fbdfb4"
  },
  "message": {
    "text": "Hello group from User",
    "msg_id": "e52fe5da281f9745ce0a"
  },
  "oa_id": "607812198688816074",
  "timestamp": "1696908900672",
  "user_id_by_app": "8109713063707884003"
}

Mô tả thuộc tính​

Thuộc tínhKiểu dữ liệuMô tả
senderJson ObjectTrong đó:
  • id: Id của User gửi tin nhắn
recipientJson ObjectTrong đó:
  • id: Id của nhóm
event_nameStringXác định loại event, gồm các giá trị sau:
  • event_name = oa_send_group_text: OA gửi tin nhắn văn bản tới nhóm
  • event_name = oa_send_group_image: OA gửi tin nhắn hình ảnh tới nhóm
  • event_name = oa_send_group_link: OA gửi tin nhán liên kết tới nhóm
  • event_name = oa_send_group_audio: OA gửi tin nhắn âm thanh thới nhóm
  • event_name = oa_send_group_location: OA gửi tin nhắn vị trí tới nhóm
  • event_name = oa_send_group_video: OA gửi tin nhắn video tới nhóm
  • event_name = oa_send_group_business_card: OA gửi tin nhắn danh thiếp tới nhóm
  • event_name = oa_send_group_sticker: OA gửi tin nhắn sticker tới nhóm
  • event_name = oa_send_group_gif: OA gửi tin nhắn gif tới nhóm
  • event_name = oa_send_group_file: OA gửi tin nhắn file tới nhóm
  • event_name = user_send_group_text: Người dùng gửi tin nhắn văn bản tới nhóm
  • event_name = user_send_group_image: Người dùng gửi tin nhắn hình ảnh tới nhóm
  • event_name = user_send_group_link: Người dùng gửi tin nhắn liên kết tới nhóm
  • event_name = user_send_group_audio: Người dùng gửi tin nhắn âm thanh tới nhóm
  • event_name = user_send_group_location: Người dùng gửi tin nhắn vị trí tới nhóm
  • event_name = user_send_group_video: Người dùng gửi tin nhắn video tới nhóm
  • event_name = user_send_group_business_card: Người dùng gửi tin nhắn danh thiếp tới nhóm
  • event_name = user_send_group_sticker: Người dùng gửi tin nhắn sticker tới nhóm
  • event_name = user_send_group_gif: Người dùng gửi tin nhắn gif tới nhóm
  • event_name = user_send_group_file: Người dùng gửi tin nhắn file tới nhóm
messageJson Object
Tùy theo event_name mà message sẽ có định dạng khác nhau, chi tiết sẽ được định nghĩa bên dưới bảng này.
Trong đó:
  • text: Nội dung tin nhắn người dùng gửi trong nhóm
  • message_id: Id của tin nhắn
Ngoài ra:
  • Nếu tin nhắn được gửi là tin nhắn đa phương tiện (Hình ảnh, ảnh gif, video,...) thì sẽ có thêm thuộc tính attachments
  • Nếu trong data trả về có thuộc tính “quote_msg_id” thì tin nhắn này được gửi qua “Trả lời”
timestampStringThời điểm request được gửi, tính bằng millisecond
oa_idStringId của Official Account

Cấu trúc thuộc tính attachments​

Thuộc tínhKiểu dữ liệuMô tả
attachmentsJson ArrayCác dạng tin nhắn đa phương tiện (Hình ảnh, ảnh gif, video,...) sẽ có phần attachments chứa các thông tin chi tiết kèm theo tin nhắn đó

Cấu trúc attachments đối với từng loại tin nhắn:​

1. Tin nhắn hình ảnh

[
    {
        "payload": {
            "thumbnail": "http://t.f6.photo.talk.zdn.vn/9c2db3ccc0a223fc7ab3.jpg",
            "url": "http://f6.photo.talk.zdn.vn/9c2db3ccc0a223fc7ab3.jpg"
        },
        "type": "image"
    }
]
Thuộc tínhKiểu dữ liệuMô tả
attachmentsJson ArrayTrong đó:
  • payload: Chứa thông tin chi tiết của tin nhắn đa phương tiện
    • thumbnail: thumbnail của ảnh
    • url: đường dẫn đến hình ảnh gốc
  • type: Loại tin nhắn
