Sự kiện xóa nhãn

Khi OA xóa nhãn qua Open API, hệ thống Zalo sẽ gửi đến webhook URL đã đăng ký trên ứng dụng

  • URL: Webhook Url của ứng dụng
  • Method: POST
  • Content Type: application/json
  • X-ZEvent-Signature: mac = sha256(appId + data + timeStamp + OAsecretKey), với data là chuỗi json trả về dưới đây.

Example​

{
    "app_id": "360846524840903967",
    "oa_id": "388613280879880645",
    "user_id_by_app": "112380535248657707",
    "event_name": "remove_tag",
    "tag": {
        "name": "example"
    },
    "timestamp": "15503500848"
}

Mô tả thuộc tính​

Thuộc tính

Kiểu dữ liệu

Mô tả

app_id

String

Id ứng dụng

oa_id

String

Id của Official Account

user_id_by_app

String

Trường này chỉ có khi app_id và oa_id trên đang liên kết với nhau.

Id này có thể sử dụng cho social_api

event_name

String

Thuộc tính xác định loại event.

tag

Json Object

Ví dụ:

{
  "name": "example"
}

Trong đó:

· name: là tên của tag bị xóa

timestamp

String

Thời điểm request được gửi, tính bằng milisecond
