Lấy danh sách admin

API này cho phép bạn lấy danh sách admin của Official Account (OA) với các quyền quản lý khác nhau.

Thông tin Request​

  • HTTP
  • cURL
GET https://EXTENSION.zalo.me/v1.0/oa/admin/getlist
curl --location --globoff 'https://openapi.zalo.me/v1.0/oa/admin/getlist?data={"user_id":"186729651760683225","role_name": "MANAGER"}' \ --header 'access_token: <your_oa_accesstoken>'

Headers​

Tham sốKiểu dữ liệuBắt buộcMô tả
access_tokenstringCóToken cho phép ứng dụng đại diện OA gọi API

Query Parameters​

Tham sốKiểu dữ liệuBắt buộcMô tả
role_namestringKhôngQuyền quản lý OA, hỗ trợ lọc các giá trị: MANAGER, MODERATOR, INSIGHT_ANALYST, ADVERTISER, CUSTOMER SERVICE, AGENCY
user_idstringKhôngID của admin cụ thể cần truy xuất thông tin
Thông tin:

Tham khảo danh sách các quyền quản lý OA tại đây.

Response​

Cấu trúc Response​

{
  "data": [
    {
      "admin_id": "186729651760683225",
      "role_name": "MANAGER"
    }
  ],
  "error": 0,
  "message": "Success"
}

Các trường trong Response​

TrườngKiểu dữ liệuMô tả
dataarrayDanh sách tất cả admin thỏa điều kiện truyền vào
data.admin_idstringID của admin
data.role_namestringPhân quyền của admin
errornumberMã lỗi (0 nếu thành công)
messagestringThông báo kết quả

Ví dụ​

  • Request
  • Response
curl --location --globoff 'https://openapi.zalo.me/v1.0/oa/admin/getlist?data={"role_name": "MANAGER"}' \ --header 'access_token: <your_oa_accesstoken>'
{
  "data": [
    {
      "admin_id": "186729651760683225",
      "role_name": "MANAGER"
    }
  ],
  "error": 0,
  "message": "Success"
}
Mẹo

Sử dụng API này cùng với API Truy xuất danh sách người dùng để lấy thêm thông tin chi tiết của admin.
