Upload ảnh

API upload ảnh template ZNS cho phép tải lên ảnh/logo trên hệ thống Zalo. Sau khi tải lên thành công, hệ thống sẽ trả về một media_id để có thể đính kèm khi tạo mẫu tin nhắn ZNS qua API tạo template ZNS (Chi tiết xem tại đây) và API chỉnh sửa template ZNS (Chi tiết xem tại đây).

Prequisite​

  • Mã truy cập (Access Token) được doanh nghiệp (OA) ủy quyền cho App.
  • Quyền cần có để sử dụng API: Quản lý tài sản.

Giới hạn​

  • File upload định dạng JPG hoặc PNG, có dung lượng tối đa 500 KB.
  • Hạn mức mỗi tháng: Tối đa 5000 ảnh được upload cho mỗi ứng dụng (App).

Lưu ý:​

  • Quy định về Logo (Xem chi tiết):
    • Định dạng: PNG
    • Kích thước: 400x96 px
  • Quy định về hình ảnh (Xem chi tiết):
    • Định dạng: JPG, PNG
    • Tỉ lệ: 16:9

HTTP request​

  • URL: https://business.openapi.zalo.me/upload/image
  • Method: POST
  • Content Type: form-data/multipart
  • Response Type: text/json

Example request​

curl --location 'https://business.openapi.zalo.me/upload/image' \
--header 'access_token: <access_token>' \
--form 'file=@"/path/to/file.png"'
