Sự kiện thay đổi về chất lượng gửi của mẫu tin ZNS

Khi có sự kiện thay đổi về chất lượng của mẫu tin ZNS phát sinh theo hệ thống đánh giá, hệ thống Zalo sẽ gửi đến Webhook Url của ứng dụng một HTTP request như sau.

Lưu ý

Quyền cần có: Sử dụng Zalo Notification Service

  • URL: Webhook Url của ứng dụng
  • Method: POST
  • Content Type: application/json
  • X-ZEvent-Signature:mac = sha256(appId + data + timeStamp + OAsecretKey), với data là chuỗi json trả về dưới đây.

Ví dụ:​

{
  "event_name": "change_template_quality",
  "oa_id": "2352868362652028400",
  "template_id": "123456",
  "quality": "HIGH",
  "timestamp": "1602560967477"
}

Mô tả thuộc tính​

Thuộc tínhMô tả
event_name
String		Tên sự kiện
Giá trị nhận về: change_template_quality
oa_id
String		ID của OA ủy quyền
template_id
String		ID của mẫu tin ZNS
quality
String		Chất lượng sau khi cập nhật.
  • HIGH – Mức độ chất lượng tốt
  • MEDIUM – Mức độ chất lượng trung bình
  • LOW – Mức độ chất lượng kém
  • UNDEFINED – Mức độ chất lượng Template chưa được xác định (trường hợp Template chưa được gửi thông báo ZNS nào trong khung thời gian đánh giá)
timestamp
String		Thời gian hệ thống gửi thông báo
